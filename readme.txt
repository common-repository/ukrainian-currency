Курс валют от НБУ для Вашего сайта.

== Description ==

<p>
Плагин Курс валют UAH поможет Вам создать блок с курсом валют относительно НБУ.
</p>

<p>
Механизм работает на основе API от Национального банка Украины. Сдесь Вы можете найти детальную информацию об курсах валют <a href="https://bank.gov.ua/control/uk/publish/article?art_id=38441973">https://bank.gov.ua/control/uk/publish/</a>
</p>

<p>
Этот плагин есть частью нашего основного API, с помощью которого, Вы можете создать блок с курсами валют и размещать его на любом сайте путем вставки сгенерированного кода. Детали здесь: <a href="https://damvb.org/kurs-informer/">https://damvb.org/kurs-informer/</a>
</p>

<p>
Плагин Курс валют UAH, поможет Вам получить свежую информацию об курсах валют и поделиться ею с пользователями Вашего сайта.
</p>

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Курс валют UAH'
3. Activate the plugin from your Plugins page.

= From WordPress.org =

1. Download 'Курс валют UAH'.
2. Upload the 'Курс валют UAH' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Курс валют UAH' from your Plugins page.
